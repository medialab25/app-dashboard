How to Use These Icons

Automatic Matching: The system will automatically match your app name to the closest icon. For example, "My Plex Server" will use the Plex icon.

Category Fallbacks: If no direct match is found, the system will use category-based fallbacks (e.g., "media" category apps get the Plex icon).

Default Fallback: If no match is found, the Docker icon will be used as the default.

Configuration: You can still specify an icon name in your app configuration, but it's not required.
